
Features of Library Management System
1.  Add new book:users can add book titles and authors.
2. Store upto 100 books:books are stored using arrays.
3. Book Availability Tracking:each bookis marked as available or browwed
4. view all books:Displays details of all books in the library
5. Search book by title:users can search books using a keyword
6. Add new memberrs:users can register new library memebers
7. store up to 50 members:member information store using arrays
8. Track borrowed book count per member:keeps count of books each member borrowed
9. Borrow books:Allows returing a borrowed book
10. Return books:allows returing a borrowed book
11. check book status:Shows whether a book is available or not
12. view all members:Display all registered member details.
13. view statistics: shows book and member counts
14. shows Available vs Borrowed Bookks:counts how manybooks are free or borrowed.
15. Most active member display:shows which member borrowed the most books.
16. input validation:checks for invalid boook or member IDs
17. Menu-Driven System :Easy navigation using numbered menu options.
18. clear Console for Clean UI : Uses Console.Clear() for neat screens.

19. Pauses After Actions :Asks user to press a key to continue.

20. Exit Option :Allows user to safely close the program.
