Features of Library Management System
1.Add new book:users can add book titles and authors.
2.Store upto 100 books:books are stored using arrays.
3.Book Availability Tracking:each bookis marked as available or browwed
4.view all books:Displays details of all books in the library
5.Search book by title:users can search books using a keyword
6.Add new memberrs:users can register new library memebers
7. store up to 50 members:member information store using arrays
8.Track borrowed book count per member:keeps count of books each member borrowed
9.Borrow books:Allows returing a borrowed book
